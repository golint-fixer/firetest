firetest

Firebase test server meant for use in unit tests

*Note: This project is not meant to be used as a Firebase replacement nor to compete with Firebase. *

Implemented

  • Basic API Usage
    • POST
    • GET
    • PUT
    • PATCH
    • DELETE
  • Query parameters:
    • auth
  • Streaming

Not Supported

  • Query parameters:
    • shallow
    • print
    • format
    • download
  • Priorities
  • Server Values
  • Security Rules
  • Error Conditions
